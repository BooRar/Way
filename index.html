Welcome to Way Android.

This Software is the development build for a commercial project. The Way project is a smart software project that combine's light Geo location software with a very smart server service to control interaction between Devices and server.

Software will be made available through the Google Play Store

Server Registration

We have decided to make the entry point of the application on the way server Way Register. From here on you can install the application with the link from Google play. Just search for the Way Family Locator.
